# Support

## How to file issues and get help

This project uses GitHub issues to track bugs and feature requests. Please search the
existing issues before filing new issues to avoid duplicates. For new issues, file your
bug or feature request as a new issue.

For help and questions about using this project:

- 📖 Read the [documentation](docs/index.md) — start with the
  [Troubleshooting & FAQ](docs/troubleshooting.md) for common issues.
- 💬 Ask questions in the repository **Discussions**.
- 🐛 Found a bug or have a feature idea? Open a GitHub **issue**.

## Microsoft Support Policy

This toolkit is provided **as-is**, without any express or implied warranties. It is a
community/field tool, **not** an officially supported Microsoft product. The underlying
Azure products and APIs it reads from are supported through standard Microsoft support
channels.

Support for this project is limited to the resources listed above.
