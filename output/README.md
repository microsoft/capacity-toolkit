# output/

Generated artifacts land here (CSV, Markdown, HTML dashboard, `capacity-config.json`,
support-ticket drafts). **This folder is intentionally empty in the shared toolkit.**

> ⚠️ Everything written here is **live tenant data** — subscription names, GUIDs,
> SKU footprint, quota numbers. **Clear this folder before sharing the toolkit** (see README §8):
>
> ```powershell
> Remove-Item .\output\* -Recurse -Force -Exclude README.md
> ```
